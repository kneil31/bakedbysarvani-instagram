# CLAUDE.md — BakedBySarvani Portal

## Overview

Public portfolio website for @bakedbysarvani (custom cakes in Dallas, TX).
Single-file static HTML generated by Python. No encryption, no frameworks.

- **Generator:** `generate_portal.py` → `output/index.html`
- **Domain (planned):** `bakedbysarvani.rsquarestudios.com`

## Architecture

- **Static HTML at build time** — all content (gallery cards, review cards, order section) rendered by Python. Crawlable by search engines without JS.
- **JS is minimal** — tab switching and category filter pills only.
- **No encryption** — public business portfolio.
- **CSP enforced** — nonce-based `script-src`, strict `connect-src 'none'`.
- **URL allowlist** — all external URLs validated server-side at build time.
- **`html.escape()`** — all untrusted text (reviews, gallery titles) escaped in Python.

## Data Sources

| File | Purpose |
|------|---------|
| `gallery_manifest.json` | Gallery card data (title, category, url, cover) |
| `../reviews.json` | Customer reviews (auto-deduplicated at build time) |
| `.secret` | Optional WhatsApp number (line 1, gitignored) |

## Tabs

1. **Gallery** — Category filter pills + photo grid tiles (SmugMug links)
2. **Reviews** — Customer testimonials with star ratings + review form CTA
3. **Order** — How to order steps + Instagram DM + WhatsApp buttons

## Quick Commands

```bash
cd portal/
python3 generate_portal.py            # Generate + open in browser
python3 generate_portal.py --no-open  # Generate only
```

## Adding Gallery Entries

Edit `gallery_manifest.json`. Each entry needs:
- `title` — Display name
- `category` — Filter pill ID (birthday, babyshower, wedding, interactive, chocolate, custom)
- `url` — SmugMug gallery URL (HTTPS only)
- `cover` — SmugMug cover image URL (optional, XLarge size recommended)
- `icon` — Emoji fallback when no cover image
- `description` — Short subtitle

Entries without a `url` are skipped at build time.

## Security

- No secrets in generated HTML (except opted-in WhatsApp number)
- `.secret` is gitignored
- External links use `target="_blank" rel="noreferrer noopener"`
- CSP blocks inline scripts, external fonts, frames, and fetch
